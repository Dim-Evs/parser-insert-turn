# parser-insert-turn

Парсер артикулов режущих пластин для токарной обработки. Добавляет в Excel-файл `parser_insert_turn.xlsx` вычисляемые колонки на листе **Parser**; справочники хранятся на листе **Map**.

## Требования

- Node.js 14+
- Microsoft Excel (для работы с `.xlsx`)

## Установка

```bash
npm install
```

## Обновление файла

Закройте `parser_insert_turn.xlsx` в Excel, затем:

```bash
npm run update:re
```

Обновляет только колонку `re` через Excel (сохраняет размер файла и ширину столбцов).

Полное обновление всех формул (может сильно увеличить размер `.xlsx`):

```bash
npm run update
```

## Структура листа Parser

| Колонка | Источник | Описание |
|---------|----------|----------|
| `art` | ввод | Полный артикул, напр. `CNMG120808-YF-GM3220` |
| `geo` | `art` | Первые 4 символа |
| `geo_r` | `art` | Первые 2 символа + `**` |
| `razmer` | `art` | Символы 5–6 |
| `S` | `art` + Map | Толщина по коду (VLOOKUP) |
| `re` | `art` + Map | Радиус/тип по символам 9–10 (VLOOKUP) |
| `chip` | `art` | Между первым и вторым `-` |
| `alloy` | `art` | После второго `-` |
| `shape` | 1-й символ `art` | Форма пластины |
| `m1`–`m6` | `alloy` (G/R) | Материал обработки по 2-й букве сплава |
| `Material_plastiny` | 3-й символ `alloy` | CVD / PVD |
| `Konstruktciia_plastiny` | 4-й символ `art` | Конструкция пластины |

## Правила парсинга

### shape — 1-й символ артикула

| Буква | Значение |
|-------|----------|
| V | Ромб 35° (V***) |
| F | Ромб 50° (F***) |
| D | Ромб 55° (D***) |
| E | Ромб 75° (E***) |
| C | Ромб 80° (С***) |
| M | Ромб 86° (M***) |
| W | Ломаный треугольник 80° (W***) |
| T | Треугольник 60° (T***) |
| R | Круг (R***) |
| S | Квадрат 90° (S***) |
| H | Шестиугольник 120° (H***) |
| K | Параллелограмм 55° (K***) |
| B | Параллелограмм 82° (B***) |
| A | Параллелограмм 85° (A***) |
| Q | Восьмиугольник 135° (O***) |
| L | Прямоугольник 90° (L***) |
| P | Пятиугольник 108° (P***) |

### re — символы 9–10 артикула (лист Map, колонки R:S)

| Код | Значение re |
|-----|-------------|
| X0 | 0.04 |
| 01 | 0.1 |
| 02 | 0.2 |
| 04 | 0.4 |
| 08 | 0.8 |
| 12 | 1.2 |
| 16 | 1.6 |
| 20 | 2.0 |
| 24 | 2.4 |
| 28 | 2.8 |
| 32 | 3.2 |
| 00 | круглая пластина |
| M0 | круглая пластина |

### m1–m3 — 2-й символ alloy (если alloy начинается с G или R)

| Буква | m1 | m2 | m3 |
|-------|----|----|-----|
| A | P | M | K |
| P | P | | |
| M | M | | |
| S | S | | |
| K | K | | |
| H | H | | |
| N | N | | |

### Material_plastiny — 3-й символ alloy

| Цифра | Значение |
|-------|----------|
| 1, 2, 6, 7 | CVD |
| 3, 4 | PVD |

### Konstruktciia_plastiny — 4-й символ артикула

| Буква | Значение |
|-------|----------|
| M | с отверстием/односторонний (***M) |
| G | с отверстием/двухсторонний (***G) |
| A | с отверстием/без стружколома (***A) |
| T | отверстие с зенковкой 40°–60°/односторонний (***T) |
| N | без отверстия/без стружколома (***N) |
| W | отверстие с зенковкой 40°–60°/без стружколома (***W) |
| H | отверстие с зенковкой 70°–90°/односторонний (***H) |
| X | иная (***X) |

## Пример

Артикул `CNMG120808-YF-GM3220`:

- 4-й символ **G** → `Konstruktciia_plastiny` = с отверстием/двухсторонний (***G)
- символы 9–10 **08** → `re` = 0.8
- `alloy` = GM3220 → `m1` = M, `Material_plastiny` = PVD
